# วิธีป้อนข้อมูลบริษัทให้ JARVIS

ทุกไฟล์ `.md` และ `.txt` ในโฟลเดอร์นี้ (ยกเว้นไฟล์ที่ขึ้นต้นด้วย `_` เช่นไฟล์นี้)
จะถูกโหลดเข้าเป็น "ฐานความรู้บริษัท" ที่ JARVIS ใช้ตอบคำถามเกี่ยวกับ SYSNECT

## วิธีเพิ่ม/แก้ข้อมูล
1. แก้ `company.md` หรือสร้างไฟล์ `.md`/`.txt` ใหม่ในโฟลเดอร์นี้ (เช่น `hr.md`, `products.md`, `faq.md`)
2. บันทึกไฟล์
3. โหลดใหม่โดยไม่ต้องรีสตาร์ท: เรียก `POST http://localhost:8000/api/knowledge/reload`
   หรือรีสตาร์ท backend

## หมายเหตุ
- ไฟล์ที่ขึ้นต้นด้วย `_` หรือ `.` จะไม่ถูกโหลด (ใช้เก็บโน้ต/คำแนะนำได้)
- รวมทุกไฟล์ถูกจำกัดขนาดที่ `KNOWLEDGE_CHAR_LIMIT` (ค่าเริ่มต้น ~12000 ตัวอักษร)
  เพื่อไม่ให้เกิน context ของโมเดล — ถ้าข้อมูลเยอะให้เพิ่มค่าใน .env
- อย่าใส่รหัสผ่าน/ความลับลงในไฟล์เหล่านี้ (AI จะเห็นและอาจเผลอพูดออกมา)
- ดูรายการไฟล์ที่โหลดอยู่: `GET http://localhost:8000/api/knowledge`
